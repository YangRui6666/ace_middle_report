# dipan_f405

这是一个基于 STM32F405 和 FreeRTOS 的底盘控制工程。

当前工程目标很明确：接收 DT7 遥控器输入，计算四轮底盘目标转速，并通过 CAN1 驱动 4 个 M3508 电机。

## 硬件接口

- `USART3`：接收 DT7 遥控器数据
- `CAN1`：收发底盘电机反馈和控制电流
- 主控：`STM32F405`

## 软件结构

工程使用 FreeRTOS 进行任务管理，当前主要有两个任务：

- `ctrlTask`
  负责读取遥控器数据，并更新底盘运动目标。
- `motorTask`
  负责读取电机反馈，执行速度环计算，并发送四个电机的目标电流。

用户代码主要放在 `User/` 目录下，按职责拆分为四层：

- `User/app`
  任务入口和任务初始化。
- `User/bsp`
  底层外设接口，包括 DBUS 和 CAN。
- `User/module`
  功能模块，包括底盘运动学、电机控制和 PID。
- `User/config`
  工程参数配置。

## 当前功能

当前版本已经完成：

- DT7 遥控器接收与解码
- 四轮底盘目标速度计算
- M3508 电机反馈解析
- 四电机速度环控制
- CAN1 电流发送

## 说明

当前代码以“先跑通底盘主链路”为目标，重点保证结构清晰、职责单一，方便后续继续接入保护、调参、方向映射和更多模式。
